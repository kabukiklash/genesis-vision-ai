# Provedores de IA - Genesis Vision

**Estado atual:** Usa apenas **Lovable** (`LOVABLE_API_KEY`).

## Configuração

No Supabase (ou Lovable Cloud) → Edge Functions → Secrets:

| Secret | Uso |
|--------|-----|
| `LOVABLE_API_KEY` | Chave da API Lovable. Obrigatório para "Gerar Código". |

Chave em: https://lovable.dev → conta → API / Integrations.
